Sofia hat einen sehr stressigen Monat hinter sich und beschließt, eine Woche Urlaub zu machen. Um Stress während ihres Urlaubs zu vermeiden, möchte sie E-Mails mit mit einer stressigen Betreffzeile an Stephen weiterleiten.

Die Funktion sollte erkennen, ob eine Betreffzeile stressig ist. Eine stressige Betreffzeile bedeutet, dass alle Buchstaben in Großbuchstaben geschrieben sind, und/oder die gesamte Zeile mit mit mindestens 3 Ausrufezeichen endet, und/oder mindestens eines der folgenden Wörter enthält "rote" Wörter: "Hilfe", "so schnell wie möglich", "dringend". Jedes dieser "roten" Wörter kann auf verschiedene Weise buchstabiert werden buchstabiert werden - "HELP", "Hilfe", "HeLp", "H!E!L!P!", "H-E-L-P", sogar auf eine "HHHEEEEEEEEELLP", sie dürfen nur keine anderen Buchstaben dazwischen haben.

Eingabe: Betreffzeile als String.

Ausgabe: Boolescher Wert.
